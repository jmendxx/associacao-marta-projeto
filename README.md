# 📚 Biblioteca Educacional - Associação Marta

## 📖 Sobre o Projeto

A Biblioteca Educacional é uma plataforma desenvolvida como parte do Projeto UPX com o objetivo de disponibilizar conteúdos educativos voltados para mulheres, jovens e famílias atendidas pela Associação Marta.

O sistema reúne materiais informativos sobre prevenção da violência, relacionamentos saudáveis, autonomia financeira, saúde mental e capacitação profissional, promovendo o acesso à informação de forma simples, organizada e acessível.

---

## 🎯 Objetivo

Criar um ambiente digital que contribua para:

- Conscientização sobre relacionamentos saudáveis;
- Prevenção da violência contra a mulher;
- Educação financeira;
- Desenvolvimento da autonomia pessoal e profissional;
- Promoção do bem-estar emocional;
- Capacitação por meio de conteúdos educativos.

---

## 🚀 Funcionalidades

### Biblioteca de Conteúdos

- Organização dos conteúdos por categorias;
- Sistema de pesquisa;
- Filtros por tema;
- Navegação entre conteúdos relacionados.

### Quiz Educativo

- Questionário interativo;
- Identificação de possíveis sinais de relacionamentos abusivos;
- Resultado personalizado de acordo com as respostas.

### Conteúdos Educacionais

- Textos explicativos;
- Vídeos complementares;
- Materiais de apoio;
- Sugestões de oficinas e capacitações;
- Temas relacionados para aprofundamento.

### Navegação

- Breadcrumb para localização do usuário;
- Botões de conteúdo anterior e próximo;
- Botão para retorno ao topo da página.
